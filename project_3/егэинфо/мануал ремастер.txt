Руководство по установке и использованию Kali Linux в VirtualBox для взлома Wi-Fi сетей с помощью Wifite2
1. Установка VirtualBox и Kali Linux
Скачайте и установите VirtualBox с официального сайта:
https://www.virtualbox.org/wiki/Downloads

Загрузите образ Kali Linux для виртуальных машин:
https://www.kali.org/get-kali/#kali-virtual-machines
(Выберите подходящую версию Kali Linux для VirtualBox.)

Скачайте Wifite2 (программа для взлома Wi-Fi):
https://github.com/derv82/wifite2

2. Настройка VirtualBox и запуск Kali Linux
Запустите VirtualBox и нажмите "Добавить машину".
В открывшемся окне выберите путь к папке, куда вы распаковали архив с Kali Linux, и выберите файл образа.

Подготовьте сетевой адаптер к режиму мониторинга:

В VirtualBox выберите вашу виртуальную машину (ВМ).

Перейдите в "Настройки" → "USB".

Справа нажмите на кнопку с портом и зелёным плюсом ("Добавить фильтр").

Выберите ваш сетевой адаптер из списка.

Запустите виртуальную машину (большая зелёная стрелка).

Логин: kali

Пароль: kali

3. Настройка сетевого адаптера в Kali Linux
После загрузки рабочего стола Kali Linux, в верхнем меню выберите "Устройства" и подключите ваш сетевой адаптер. (нажмите на него)

Откройте терминал (иконка в левом верхнем углу, похожая на командную строку) и введите команду:

lsusb

В списке USB-устройств найдите ваш сетевой адаптер.

Проверьте доступные интерфейсы:

sudo airmon-ng

Вы увидите список интерфейсов. Например:

PHY     Interface       Driver          Chipset
phy0    wlan0           ath9k_htc       Qualcomm Atheros Communications
Переведите интерфейс в режим монитора:

sudo airmon-ng start wlan0

После выполнения команды интерфейс будет переименован (например, wlan0 станет wlan0mon).

Остановите мешающие процессы (если есть):

sudo airmon-ng check kill

Затем повторите команду:

sudo airmon-ng start wlan0

Проверьте режим монитора:

iwconfig

Убедитесь, что интерфейс (например, wlan0mon) теперь находится в режиме Monitor.

4. Перенос Wifite2 на виртуальную машину
Добавьте папку с программой Wifite2 в настройки виртуальной машины:

Выберите виртуальную машину и нажмите "Настроить".

Перейдите во вкладку "Общие папки" и нажмите на значок папки с плюсом.

Выберите папку с Wifite2 и поставьте галочку напротив "Авто-подключение".

Нажмите ОК.

В Kali Linux откройте файловый менеджер и найдите папку с разархивированной программой Wifite2.
Перетащите её на рабочий стол.

5. Запуск Wifite2
На рабочем столе Kali Linux нажмите правой кнопкой мыши по папке с Wifite2 и выберите "Открыть с правами администратора".

Найдите файл Wifite.py, нажмите на него ПКМ и выберите "Открыть терминал здесь".

В терминале введите команду:

python Wifite.py

Программа предложит скачать дополнительные инструменты.
Рекомендация: Скачайте их, так как они повышают эффективность программы. Однако, авторы этих дополнений мудаки, не оптимизировали код для Python 3, поэтому
сли захотите скачивать их, то вам, придётся вручную исправлять ошибки в коде.
#сегодня 14.02.2025 и пока они ничего не изменили
(Ссылки на обновления вы увидите в программе.)

6. Взлом Wi-Fi сети
Программа предложит выбрать сеть для взлома.

Нажмите Ctrl + C, чтобы остановить сканирование.

Введите номер сети, которую хотите взломать.

Начнётся процесс взлома. Дождитесь его завершения.
Если взлом не удался, попробуйте выбрать другую сеть.

Примечание:
Автор мануала: Artipopik.
Без дополнительных инструментов автор смог взломать 2 из 5 сетей своих соседей.
